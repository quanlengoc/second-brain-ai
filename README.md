# 🧠 Second Brain AI

> **Build an AI-powered Second Brain while becoming an AI Engineer.**

Second Brain AI is not just another AI application.

It is a long-term learning project designed to combine **AI Engineering**, **English improvement**, and **professional certification preparation** into a single real-world product.

Instead of learning technologies in isolation, every concept is immediately applied by building a practical AI system.

---

# 🎯 Mission

This project has three primary goals.

## 1. Earn AI Certifications

Complete AI learning paths available through Coursera Plus and transform theoretical knowledge into working software.

Examples include:

* AI Fundamentals
* Prompt Engineering
* Retrieval-Augmented Generation (RAG)
* AI Agents
* Model Context Protocol (MCP)
* LLM Application Development

---

## 2. Prepare for PMI-PBA

Develop business analysis skills while designing and documenting a real software product.

The project follows professional software engineering practices including:

* Requirements Analysis
* Architecture Design
* Documentation
* Version Control
* Project Planning

---

## 3. Improve English (B1 → B2)

English is used throughout the entire project.

Instead of studying English separately, it becomes part of the daily workflow.

Activities include:

* Reading technical documentation
* Watching Coursera lectures
* Learning domain vocabulary
* Writing journals
* Creating architecture documents
* Writing Git commits and pull requests

---

# 💡 Learning Philosophy

Every lesson follows the same cycle.

```text
Learn
    ↓
Understand
    ↓
Document
    ↓
Build
    ↓
Reflect
    ↓
Share
```

Knowledge is only considered complete after it has been implemented in the project.
